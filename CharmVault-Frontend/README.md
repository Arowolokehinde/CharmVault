# CharmVault Frontend

Programmable Bitcoin Vaults - Built on Charms Protocol

## Design System

- **Dark Theme**: Bitcoin-native, protocol-grade UI
- **Glassmorphism**: Soft glow accents, premium aesthetic
- **Colors**: Teal, Amber, Purple accents
- **Typography**: Inter + Space Grotesk

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Framer Motion
